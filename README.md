This project is an open competition and benchmark for LLM-based quantitative trading agents. Entrants build agents that autonomously mine factors, train predictive models, and produce trading strategies, all scored on a common dataset and backtesting pipeline so results are directly comparable. The goal is a transparent, reproducible leaderboard for "alpha from LLMs" — and a shared reference point for researchers and practitioners in the field. Status: early groundwork. The design draws inspiration from Microsoft's RD-Agent(Q) (arXiv:2505.15155); see acknowledgements.
